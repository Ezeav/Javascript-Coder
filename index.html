Simulador Interactivo

Este simulador permite realizar distintas acciones interactivas desde la consola y a través de cuadros de diálogo. El usuario puede validar su edad, ingresar colores, calcular impuestos según la marca de su auto y realizar operaciones matemáticas básicas.

El proyecto está realizado en JavaScript puro y utiliza condicionales, ciclos, funciones, arrays y objetos, así como el uso de prompt, confirm y alert para interactuar con el usuario.
